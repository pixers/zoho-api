# Zoho API Wrapper

Library offers API wrapper for Zoho CRM (https://www.zoho.com).

## Installation

The preferred way to install the library is using [composer](http://getcomposer.org/).

Run:

```bash
php composer.phar require "pixers/zoho-api"
```
